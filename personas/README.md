# Personas

Custom Persona Lab profiles are saved here by default:

```text
personas/presets_custom.json
```

The real custom file is user-owned runtime data at `%LOCALAPPDATA%\VoiceGenOui\user-personas\presets_custom.json` and is outside the repository. It is ignored by git when manually copied into a checkout. Use `presets_custom.example.json` as a starting shape for sharing or manual editing.
